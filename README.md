## Related Backend Repo

https://github.com/bonzoyang/buyitcrashit

### Quick Start

```
git clone git@github.com:bonzoyang/buyitcrashit.git
cd ./buyitcrashit/docker
docker-compose build
docker-compose up 
```

backend server will run on localhost:8000
